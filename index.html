ML Fraud & Anomaly Detection

🔍

Project Overview

Developed an ML-based anomaly detection framework for EHR access logs to identify unauthorized access and insider threats using multiple supervised and unsupervised learning approaches.

Key Highlights

  • Implemented Isolation Forest (unsupervised), Random Forest, XGBoost (supervised), and Autoencoder neural network models
  • Built a comprehensive preprocessing pipeline: missing values, categorical encoding, feature normalization, and behavioral attributes
  • Engineered features like login frequency, failed login attempts, and after-hours access detection
  • Evaluated with Accuracy, Precision, Recall, F1-Score, and Confusion Matrix across all models

Tech Stack

PythonScikit-learnXGBoostTensorFlow/KerasPandasNumPy
