CPU Scheduling Simulator

⚙️

Project Overview

A web-based application that demonstrates and visualizes how different CPU scheduling algorithms work in an OS. Users enter process details (Arrival Time, Burst Time, Priority) and observe scheduling through dynamic Gantt charts and automatic metric calculations.

Algorithms Implemented

  • FCFS — First Come First Serve (non-preemptive, order of arrival)
  • SJF — Shortest Job First (minimizes average waiting time)
  • SRTF — Shortest Remaining Time First (preemptive SJF)
  • Round Robin — Fair CPU allocation with fixed time quantum
  • Priority Scheduling — Preemptive & Non-Preemptive variants

Key Features

  • Dynamic Gantt Chart with color-coded process visualization
  • Auto-calculated metrics: CT, TAT, WT, Avg TAT, Avg WT
  • Interactive process input — add processes dynamically
  • Algorithm comparison across all scheduling techniques
  • Supports arrival times and Round Robin queue management

Tech Stack

HTMLCSSJavaScriptDOM Manipulation
